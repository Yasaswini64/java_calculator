# Java Calculator
A simple calculator implemented in Java with support for basic arithmetic operations, power, and square root functionality.

## Features
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulus (`%`)
- Power (`^`)
- Square Root (`#`)

## How to Run
- Compile the program: `javac Calculator.java`
- Run the program: `java Calculator`

## Technologies Used
- Java
